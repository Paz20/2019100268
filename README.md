
# Turtle Unida



## Comandos para iniciar

**1.** Iniciar ROS: 



```bash
  roscore 
```

**2.** Iniciar la tortuga: 



```bash
  rosrun turtlesim turtlesim_node
```


**3.** Ejecutar el siguiente comando en una nueva terminal: 



```bash
  devel\setup.bat
```

**4.** Para mover la tortuga, ejecutar: 



```bash
  rosrun turtle_unida src/mover.py
```
